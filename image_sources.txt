Images for this dataset were sourced from many locations around the web.
The bulk of each set was scraped from https://spiderid.com/
The web-scraping program I wrote for SpiderID.com is included in this directory (spiderid_scrape.py).

Additional images were sourced from the following online databases:
https://bugguide.net/node/view/1954
https://www.insectidentification.org/spiders.php
https://www.insectimages.org/browse/taxthumb.cfm?order=12
